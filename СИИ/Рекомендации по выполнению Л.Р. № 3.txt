В архиве методичка по 3-ей лабораторной работе и Turbo Prolog. В методичке приведены методические сведения
и пример программы на Прологе, для предметной облсти "Породы собак". Он также находится в файле Prim.pro.
По аналогии с ним, вы должны построить прототип экспертной системы на Прологе для своей предметной области (Л.р. № 1). 

Если проблемы с запуском программы то нужно использовать Dosbox.
Работа с Dosbox:
1. Скачать Dosbox (http://www.dosbox.com/download.php?main=1), там есть зеленая кнопка "Download Now".
2. Установить Dosbox
3. Скопируйте TProlog на c:\TProlog
4. В консоли Dosbox наберите:
	mount q c:\TProlog
	q:
	start
5. Откройте Prim.pro (File->Load). Это отправная точка для выполнения Лабораторной работы № 3.
6. Запустите программу на выполнение командой Run (Alt+R).
